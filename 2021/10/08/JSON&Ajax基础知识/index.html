JSON 基础知识 & Ajax 基础知识

为什么我们要把对象转化为字符串？

  1. 在网页开发的时候，我们需要把对象信息进行长期保存, 因此我们需要把这些信息传递给服务器。所以，需要通过把对象(变量)转化为字符串才能传递给服务器。

  2. 我们采用的是http协议即超文本传输协议(其本质就是由文本的形式进行传输)

  3. JSON 字符串是绝大多数编程语言都支持的…

JSON 字符串的特征

  1. 属性名称必须加双引号 (因为人家默认规定的字符串外层是单引号，所以内层必须是双引号)
  2. 属性值如果是字符串则必须加双引号
  3. 对象当中的方法(函数)不会被转化为字符串

(注意： JSON 本质就是字符串！！)
