  • 创建各种随机的 id: uuid

  • 临时 email(temporary email): temp-mail
    (方便测试邮箱登录、修改密码等功能)

  • 免费实现用户认证 firebase：firebase

  • 在 react 中使用方便的 icon：fontawesome

  • 提供假的数据fake Api

  • 方便接口的 url 传入参数： qs

  • 字体下载网站： fonts

  • 处理时间格式的库：dayjs

  • react 错误边界库：react-error-boundary

  • react 头部标题控制：react-helmet

  • 查找 react 中无限渲染的原因的库：why-did-you-render

  • 获取免费的图片: unsplash

  • 获取 icon 的网站: font awesome (这个网站有免费的 icon，然后下载它放在自己项目文件里就可以直接使用了…)

  • 实现语言国际化 i18n
