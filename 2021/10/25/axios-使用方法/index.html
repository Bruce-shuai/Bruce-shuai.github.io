axios 使用方法

官方文档
Axios 是一个基于 promise 网络请求库，既可以作用于 node.js 也可以作用于浏览器中。 它是 isomorphic 的(即同一套代码可以运行在浏览器和 node.js 中)。在服务端它使用原生 node.js http 模块, 而在客户端 (浏览端) 则使用 XMLHttpRequests。

这个不像fetch。axios 是需要在项目中额外安装的
