JSON细节知识

什么是 JSON

JSON(JavaScript Object Notation)

  • 是一种数据的交换格式(和 xml 很相似)
  • 通常被用于接口(APIs)和配置文件
  • 相较于(xml) JSON 是轻量级且容易读和写的
  • 兼容各种语言 (特别是针对 js 是兼容性最好的，JSON 实际上是 JavaScript 的一个子集)

JSON 常常用与前后端的数据交互

JSON 具备的类型

JSON 的字符串规定必须用双引号""，Object 的键也必须用双引号""

类型名 示例
Strings “Hello World” “Hi”(注意是双引号)
Numbers 10 1.5 -30 1.2e10(兼容各种进制和科学计数法)
Booleans true false
null null (注意，没有 undefined 类型)
Arrays [1, 2, 3], [“Hello”, “World”]
Objects {“key”: “value”} {“age”: 30} (注意，value 可以是 String、Number、Boolean、null、Array、Object 类型)
